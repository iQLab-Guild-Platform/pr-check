# pr-check
ギルドプラットフォーム上におけるソフトウエア開発系クエストの共通課題「GitHub の Pull Request」の提出先

## プルリクエストの内容
- submission フォルダに自身の Github アカウントの username と同じ名前のテキストファイルを作成する。
